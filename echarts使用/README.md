> #### Echarts的基本使用
>
> 官方网址：[Apache ECharts](https://echarts.apache.org/zh/index.html)
>
> script标签：`<script src="https://cdn.bootcdn.net/ajax/libs/echarts/5.1.2/echarts.common.js"></script>`
>
> npm安装：`npm install echarts`
>
> 主要步骤：
>
> 1. 准备好放置图表的容器
> 2. 基于准备好的DOM，初始化echarts实例
> 3. 指定图表的配置项和数据
